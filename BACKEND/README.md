# 🎓 AI Student Registration Assistant

## 📌 Project Overview

AI Student Registration Assistant is a Full Stack Web Application that allows users to register, edit, delete, search, and manage student records.

The project was developed to learn the complete Full Stack Development workflow using HTML, CSS, JavaScript, FastAPI, MongoDB, GitHub, and Vercel.

---

## 🚀 Features

- Student Registration
- View Registered Students
- Edit Student Details
- Delete Student
- Search Students
- Dashboard Statistics
- AI Assistant Section
- Responsive UI
- REST API using FastAPI
- MongoDB Database Integration

## ⚙ Installation

Clone the repository

git clone https://github.com/YOUR_GITHUB_USERNAME/AI_Student_Registration_Assistant.git

Go to Backend

cd BACKEND

Install packages

pip install -r requirements.txt

Run FastAPI

uvicorn main:app --reload

Open Frontend

Open index.html using Live Server.
